# Tela de Login com JavaFX e FXML

## 📌 Sobre o projeto

Este projeto foi desenvolvido com o objetivo de praticar a criação de interfaces gráficas utilizando JavaFX e FXML.

A aplicação consiste em uma tela de login estilizada com CSS, contendo campos de email e senha, botão de login e mensagens de validação utilizando Alert.

---

## 🚀 Tecnologias utilizadas

- Java
- JavaFX
- FXML
- CSS

---

## 🎨 Funcionalidades

✔️ Tela de login moderna  
✔️ Estilização com CSS  
✔️ Campos de email e senha  
✔️ Botão de login  
✔️ Mensagem de sucesso ao efetuar login  
✔️ Mensagem de erro ao deixar campos vazios  

---

## 📚 O que aprendi

Durante o desenvolvimento deste projeto, pratiquei:

- Estruturação de telas com FXML
- Estilização de interfaces com CSS no JavaFX
- Utilização de componentes como:
  - VBox
  - HBox
  - StackPane
  - TextField
  - PasswordField
  - Button
  - Alert
- Criação de eventos com `onAction`
- Uso de Controllers no JavaFX
- Integração entre FXML e Java
- Organização de projeto JavaFX
